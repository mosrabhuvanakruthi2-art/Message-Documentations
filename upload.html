Upload PDF by migration path

Choose the combination and document type, then upload a PDF. The PDF text is used to update the section content (item descriptions). Page numbers, leading section numbers (e.g. "2 •"), and bullet points (•) are stripped automatically. Screenshots are not extracted from the PDF—use Upload screenshot to add images. Use this when the document is updated with new features or changes.
Run the server with upload support: node scripts/server.js (then open this page at localhost:3000/upload.html).
