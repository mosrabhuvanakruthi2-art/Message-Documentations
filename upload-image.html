Upload screenshot for a feature

Choose the combination, document type (Supported features or Limitations), then the specific feature or limitation. Upload one image with the button, or open the project screenshot folder and add/replace images there (use the filename shown).

Select a feature to see the filename (e.g. public-channels.png). Save images with this name in the folder to replace.

Screenshots are stored in the project at assets/screenshots/<combination>/. Opening the folder lets you add or replace images: use the filename above. Replacing a file with the same name updates the screenshot after you refresh the Screenshots page.
